# Resume Rewrite Skill

## When to use
- User asks for concise resume version
- User asks for detailed resume version
- User wants safer, more defensible wording
- User wants interview-friendly bullet rewriting

## Workflow
1. Parse the resume into structured sections.
2. Preserve factual content and avoid adding unsupported claims.
3. Generate multiple rewrite styles if requested:
   - concise
   - detailed
   - conservative
   - interview-friendly
4. Keep bullets measurable only when the original text supports that measurement.
5. Return the rewritten artifacts and optionally improved interview answers.

## Output principles
- Prefer defensible claims over inflated claims.
- Keep wording concise and easy to explain in interview.
- Preserve user-provided facts and timeline.
